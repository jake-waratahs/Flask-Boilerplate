Loop Example!

{# Here we are going to loop through some data passed to us #}
    {% for item in list_data %} {# Notice how we use the percent bracket for loops and any template logic, whereas {{ }} brackets are used for expressions #}
  • {{ item[0] }} - {{ item[1] }}
    {{ item[2] }}
    • {% endfor %}
{# Exciting, that example just spat out every item in the list. We can actually do even cooler things with that list. #}

Skip The First Item:

    {# Skip the first item in the list (Python List Index Notation Magic) #} {% for item in list_data[1:] %}
  • {{ item[0] }} - {{ item[1] }}
    {{ item[2] }}
    • {% endfor %}

Impose a condition

    {# Only loop items which a condition is true for. #} {% for item in list_data if item[0] == 0 %}
  • {{ item[0] }} - {{ item[1] }}
    {{ item[2] }}
    • {% endfor %}

Show a message if no items are in the list (or matched the conditions)

    {# Only loop items which a condition is true for. #} {% for item in list_data if item[0] == 10 %}
  • {{ item[0] }} - {{ item[1] }}
    {{ item[2] }}
    • {% else %}

    Sorry, no items are in this list :(

    {% endfor %}

Filters

A Filter is a function that takes text as it's first argument, and can take optional/required parameters. They are useful for formatting text and other items within a template.

{{ lowercase_text|upper }}

{# turn the string in lowercase_text uppercase. #}
