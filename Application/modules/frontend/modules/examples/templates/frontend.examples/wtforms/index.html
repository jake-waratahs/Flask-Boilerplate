WTForms

HTML forms are really difficult to get right. Validation ends up having non-reusable nested if statements, and gets really messy to read and understand the logic. For this reason, WTForms makes forms on the web 100x easier.

WTForms lets you generate a form from a Python class. Links to some examples are included below:
